    字数:239 阅读 :
    1. 镜像
    2. conda配置

    镜像

    1. 拉取系统镜像pytorch，下载到个人文件夹，名为pytorch.image

    2. 点击导入，选择这个镜像，重命名为binary-q，创建后可以看到它的存储位置：/dssg/home/scs2010813971/.lico/container/binary-q.sif

    1. 命令行进入该路径后，完成初始配置。
    $ singularity shell binary-q.sif

$ pip install einops bayesian-optimization --user

    conda配置

    命令后进入/dssg/home/scs2010813971/.lico/container/，操作镜像，进行conda虚拟环境和相关包的安装

    $ singularity shell binary-q.sif
Singularity> conda create -n bq python=3.6
Singularity> source activate bq
(bq) Singularity> conda deactivate
Singularity> conda activate bq
(bq) Singularity>

    目前conda base的环境中（/opt/conda/bin/conda）已经安装了很多包了，而且与其他用户应该是独立的状态，所以可以直接在base上安装（目前不清楚虚拟环境要配置的话，要怎么选择运行时的虚拟环境？）
